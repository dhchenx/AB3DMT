##  Agent-based 3D Modeling Toolkit

This toolkit aims to support complex 3D simulation with agent-based models in NetLogo. 

### Functions 

- Import 3D Model into NetLogo
- 3D Agent Visiting
- 3D Model Scanning
- 3D Particle Collision
- Infrared 3D Data Rendering
